# Video Recording Guide - Chess Game Demo

**Duration:** 60-90 seconds
**What to Show:** Game menu → gameplay → winning position

---

## Using Windows Built-in Screen Recorder (Easiest)

### Step 1: Open Game
```
1. Press Windows + R
2. Type: powershell
3. Paste: cd "C:\Users\PC\Desktop\mini project c" ; .\chess.exe
4. Press Enter - game starts
```

### Step 2: Start Recording
```
1. Press Windows + G (opens Game Bar)
2. Click "Start recording" (red circle button)
   - Or press Windows + Alt + R
3. You'll see "Recording started" notification
```

### Step 3: Play the Game (What to Demo)
**60-90 seconds total - follow this script:**

**[0-10 sec] Show Menu:**
- Game displays menu
- Choose option 1 (Play)

**[10-15 sec] Show Starting Position:**
- Display the full 8×8 board
- All pieces in starting positions
- Show piece colors clearly (blue = white, red = black)

**[15-50 sec] Make Moves:**
```
White moves:      Black moves:
e2 e4      →      e7 e5
g1 f3      →      g8 f6
f1 c4      →      f8 c5
d2 d3      →      d7 d6
```
Wait 1-2 seconds after each move to show the board update clearly.

**[50-80 sec] Optional - Show More Gameplay:**
Continue playing or show:
- King moving to safety
- Capturing a piece
- Any interesting position

**[80-90 sec] End Recording:**
- Type `quit` to exit game
- Stop recording (Windows + Alt + R or click stop)

---

### Step 4: Save Recording
Windows saves it automatically:
- **Location:** `C:\Users\PC\Videos\Captures`
- **File name:** Something like `2026-04-07 15-30-45.mp4`

---

### Step 5: Upload Video

**Option A: YouTube (Recommended for professors)**
1. Go to youtube.com
2. Click your profile icon → Create a video
3. Click "Upload video"
4. Select the MP4 file from Videos\Captures
5. Title: "Chess Game C - Mini Project"
6. Visibility: "Unlisted" (only people with link can see)
7. Click "Publish"
8. Copy the YouTube URL (looks like: https://youtu.be/xxxxx)

**Option B: Google Drive (If no YouTube account)**
1. Go to drive.google.com
2. Click "New" → "File upload"
3. Select MP4 from Videos\Captures
4. Right-click file → "Share"
5. Change to "Anyone with the link"
6. Copy the sharing link

**Option C: Loom.com (Automatic - No Upload Needed)**
1. Go to loom.com
2. Sign up with Google
3. Click "Start recording"
4. Follow Steps 1-3 above directly in Loom
5. It uploads automatically
6. Copy the Loom link when done

---

## What NOT to Do (Common Mistakes)

❌ Don't record console output text - focus on BOARD
❌ Don't make 10+ moves - keep it 60-90 seconds
❌ Don't close game without typing `quit` - looks messy
❌ Don't record audio unless you're narrating
❌ Don't use private/restricted YouTube visibility

---

## Success Checklist

✅ Video is 60-90 seconds
✅ Game menu is visible
✅ At least 4-5 moves shown
✅ Board is clear and readable
✅ No recording artifacts or glitches
✅ Video is uploaded and link works
✅ Link is PUBLICLY ACCESSIBLE (not private)
